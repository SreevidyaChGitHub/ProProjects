# Research Library

<b>Client: </b>Merrill Lynch Inc (Bank of America), US<br/>

<b>Description</b><br/>

The Research Library is a web‑based platform used by analysts, advisors, and clients to search and access research reports. It provides fast, flexible search capabilities and allows users to view research content in various customizable formats. The application serves as a key strategic tool for the organization’s research distribution. 

Core Module:  <br/>
The core module enables users to efficiently search, filter, and retrieve research reports using advanced search technologies. It supports user profiles, personalized access, and a streamlined interface for navigating large volumes of research content.

Indices Module:  <br/>
The Indices module provides market index information, including price movements and calculated metrics. Users can view snapshot summaries, detailed index data, and visual charts that help track performance trends.
